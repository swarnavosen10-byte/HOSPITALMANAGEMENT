# Project Overview

## Project Name
MediSync – Smart Healthcare Management System

## Introduction
MediSync is a centralized healthcare platform that connects multiple hospitals and provides real-time information about hospital facilities such as available beds, doctors, ambulances, and departments.

## Objective
The objective of this project is to reduce delay during medical emergencies and help users quickly find hospitals with required resources.

## Main Features
- Real-time bed availability
- Doctor information
- Ambulance booking
- Hospital map with availability pins
- Moving ambulance tracking
